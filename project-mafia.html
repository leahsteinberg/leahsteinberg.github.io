Leah Steinberg : Project: Mafia

Sun 13 April 2014

The one problem with the game Mafia is you need someone to be 'God'. So I made a text-message-based Mafia game (so our phones may remain our gods!) I want this game to be played remotely by big groups, constantly, to keep everyone tense and checking their text messages. I used the Twilio API to send texts and Python's Flask for the server logic.
